Задание 1 - работа с pandas датафреймом (EDA анализ)
Описание полей:
- client_id - идентификатор клиента
- education - уровень образования
- sex - пол заемщика
- age - возраст заемщика
- car - флаг наличия автомобиля
- car_type - флаг автомобиля иномарки
- decline_app_cnt - количество отказанных прошлых заявок
- good_work - флаг наличия “хорошей” работы
- bki_request_cnt - количество запросов в БКИ
- home_address - категоризатор домашнего адреса
- work_address - категоризатор рабочего адреса
- income - доход заемщика
- foreign_passport - наличие загранпаспорта
- sna - связь заемщика с клиентами банка
- first_time - давность наличия информации о заемщике
- score_bki - скоринговый балл по данным из БКИ
- region_rating - рейтинг региона
- app_date - дата подачи заявки
- default - флаг дефолта по кредиту
Подготовить ноутбук. На основе данных из приложенного файла выполнить
следующие задания:
1. Посмотреть информацию о датафрейме, проверить данные на пропуски и
выбросы
2. Посмотреть зависимость между признаками
3. Проверить предложенные гипотезы и описать выводы
Гипотезы:
1. Возраст "хороших" заемщиков больше, по сравнению с "плохими" (распределения
возраста в зависимости от флага дефолта смещено в большую сторону при
default=0), возрастные более платежеспособные
2. Уровень образования зависит от возраста, что влияет и на возврат кредита, также
люди с высшим образованием чаще являются "хорошими" заемщиками
3. При good_work = 0 увеличивается риск невозврата кредита (флаг дефолта)
4. Доход "хороших" заемщиков больше, по сравнению с "плохими" (распределения
дохода в зависимости от флага дефолта смещено в большую сторону при default=0)
5. score_bki напрямую взаимосвязан с default, чем он меньше, тем выше вероятность
клиента выплатить кредит банку

6. Наличие авто влияет на вероятность возврата кредита
7. Рейтинг региона влияет на размер зарплаты, что также может повлиять на возврат
кредита

Описание для второго:
Продуктовый аналитик Рекомендации
Задание №1
Ты - аналитик продуктовой команды Главной страницы 
Руководитель попросил тебя найти точки роста эффективности главной ленты (main
feed) — того самого первого экрана с рекомендациями, который видит пользователь сразу
после открытия приложения или сайта.
У тебя есть данные только по взаимодействиям пользователей (без персональных
данных). Данные содержат информацию **как по главной ленте, так и по всей платформе
за последний месяц (~7000 пользователей).
Данные
Ты получишь два файла:
1. users.csv - агрегированные метрики по каждому пользователю (7000 строк, одна
строка = один пользователь)
Колонка Описание
user_id Уникальный ID пользователя
total_product_views Всего просмотров карточек товаров по всему маркетплейсу
total_pd_clicks Клики по карточкам → переход на карточку товара по всему маркетплейсу
total_add_to_cart Всего добавлений в корзину по всему маркетплейсу
total_purchases Всего покупок по всему маркетплейсу
total_revenue Общая сумма покупок, руб
feed_views Просмотры только в главной ленте
feed_clicks Клики только в главной ленте
feed_add_to_cart Добавления в корзину только из главной ленты
feed_purchases Покупки только из главной ленты
2. user_category_interactions.csv — сырые данные по взаимодействиям (~87 000 строк)
Колонка Описание
user_id ID пользователя
Колонка Описание
category Категория товара (Одежда, Обувь, Красота и здоровье, Электроника и
т.д.)
placement Тип размещения / ленты: • main_feed • search_results • category_page •
pdp_recommendations • homepage_carousel
views Количество просмотров
clicks Количество кликов
add_to_cart Количество добавлений в корзину
purchases Количество покупок
Важно: Метрики feed_* из первой таблицы = точная сумма строк с placement =
'main_feed' из второй таблицы.
Задача
Предложи минимум 5 гипотез по улучшению главной ленты (main_feed).
Каждая гипотеза должна содержать:
1. Чёткую формулировку (что именно и почему мы хотим изменить).
2. 1–2 максимально читаемых графика (bar chart, grouped bar, stacked bar, line и т.д.).
Опиши график текстом + ключевые цифры (можно строить в Excel, Python или
Tableau).
3. Краткое описание анализа (что видно из данных).
4. Выводы и конкретные рекомендации (что тестировать, какой ожидаемый эффект
на главную ленту и на весь маркетплейсу — revenue, purchases и т.д.).
Примеры тем для анализа (не обязательно использовать все):
Сравнение главной ленты с другими placement’ами
Разница по категориям товаров
Влияние главной ленты на общую воронку маркетплейсу
Поведение разных групп пользователей (можно самим найти сегменты)
Формат ответа:
Только текст + описания графиков (Markdown или Google Slides).
Графики должны быть простыми и очень понятными.
Фокус на actionable insights - что именно можно улучшить и как это повлияет на
бизнес.
