### Задание №1
Ты - аналитик продуктовой команды Главной страницы 
Руководитель попросил тебя найти точки роста эффективности главной ленты (main
feed) — того самого первого экрана с рекомендациями, который видит пользователь сразу
после открытия приложения или сайта.
У тебя есть данные только по взаимодействиям пользователей (без персональных
данных). Данные содержат информацию **как по главной ленте, так и по всей платформе
за последний месяц (~7000 пользователей).
Данные
Ты получишь два файла:
1. users.csv - агрегированные метрики по каждому пользователю (7000 строк, одна
строка = один пользователь)
Колонка Описание
user_id Уникальный ID пользователя
total_product_views Всего просмотров карточек товаров по всему маркетплейсу
total_pd_clicks Клики по карточкам → переход на карточку товара по всему маркетплейсу
total_add_to_cart Всего добавлений в корзину по всему маркетплейсу
total_purchases Всего покупок по всему маркетплейсу
total_revenue Общая сумма покупок, руб
feed_views Просмотры только в главной ленте
feed_clicks Клики только в главной ленте
feed_add_to_cart Добавления в корзину только из главной ленты
feed_purchases Покупки только из главной ленты
2. user_category_interactions.csv — сырые данные по взаимодействиям (~87 000 строк)
Колонка Описание
user_id ID пользователя
Колонка Описание
category Категория товара (Одежда, Обувь, Красота и здоровье, Электроника и
т.д.)
placement Тип размещения / ленты: • main_feed • search_results • category_page •
pdp_recommendations • homepage_carousel
views Количество просмотров
clicks Количество кликов
add_to_cart Количество добавлений в корзину
purchases Количество покупок
Важно: Метрики feed_* из первой таблицы = точная сумма строк с placement =
'main_feed' из второй таблицы.
Задача
Предложи минимум 5 гипотез по улучшению главной ленты (main_feed).
Каждая гипотеза должна содержать:
1. Чёткую формулировку (что именно и почему мы хотим изменить).

Tableau).
3. Краткое описание анализа (что видно из данных).
4. Выводы и конкретные рекомендации (что тестировать, какой ожидаемый эффект
на главную ленту и на весь маркетплейсу — revenue, purchases и т.д.).
Примеры тем для анализа (не обязательно использовать все):

Разница по категориям товаров
